- Отмена: за **24 часа** до приёма

# Памятка: ДМС

Приём по ДМС: полис + гарантийное письмо + паспорт.

Партнёры: СОГАЗ (8-800-333-08-88), Ингосстрах (8-495-956-55-55), РЕСО-Гарантия (8-495-730-30-00), АльфаСтрахование (8-800-333-05-41), ВТБ Страхование (8-800-100-44-03).

**Покрывается (типовая программа):** консультации, лечение кариеса, рентген, профгигиена (1-2 р/год), удаление, лечение дёсен.

**Не покрывается:** имплантация, протезирование, ортодонтия, эстетика (виниры, отбеливание).

Конкретные лимиты — по полису. **Оператор НЕ подтверждает покрытие**, только страховая.

Порядок: проверить страховую в списке → пациент получает гарантийное письмо → запись с пометкой "ДМС" → на приёме проверка документов → счёт страховой.

Без гарантийного письма — платный приём. ДМС и акции не суммируются.
